# 一些正则匹配示例

## facebook

```
(^https://.*\.fbcdn\.net/.*)&bytestart=.*
```

## YouTube

```
(^https://.*\.googlevideo\.com\/videoplayback\?.*)&range=.*
```

扩展填写MP4

## instagram

```
(^https://scontent[a-z0-9-]*\.cdninstagram\.com/v/.*)&bytestart=.*
```

如果你有超棒的正则 可以在github或用mail分享出来。
