# 猫抓(cat catch)

猫抓(cat-catch) 资源嗅探扩展，能够帮助你筛选列出当前页面的资源。提供嗅探 缓存捕捉 视频录制等各种工具帮助抓取资源。可以在设置内添加想要抓取的资源类型，包括图片等资源。
